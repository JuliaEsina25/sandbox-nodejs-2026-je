 Решение квадратных уравнений на JavaScript

Программа для вычисления действительных корней квадратного уравнения **ax² + bx + c = 0**, написанная на JavaScript для Node.js.

- Node.js (V 24.)

Требования к программе, вычисляющей корни квадратного уравнения:

0. при отсутствии аргументов программа должна выводить справку  в stdout и выходить с кодом результата 1.
1. ВВод данных через аргументы (a, b, c); 
2. Вывод данных в случае нахождения корней уравнения в stdout с кодом результата 0;
3. Вывод данных в случае ошибки (корней нет) в stderr с кодом результат на выбор (не 0 и не 1); 
4. Подготовить Readme с описанием эксперимента по запуску программы с аргументами, демонстрирующими все режимы работы.     

Режим работы: 
1. Корней нет. 
2. Один корень. 
3. Два корня.


Установка и запуск

1. Создан файл index.js с кодом программы
2. Создан файл README.txt с документацией
3. Тестовые команды:

Тест справки: 
node index.js

Тест без корней: 
node index.js 1 0 1
echo $?  # Должно быть 2

Тест с одним корнем
node index.js 1 -2 1
echo $?  # Должно быть 0

Тест с двумя корнями
node index.js 1 -3 2
echo $?  # Должно быть 0

Тест ошибки валидации
node index.js 1 2
echo $?  # Должно быть 1
